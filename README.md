# CreditWise Loan System

A machine learning-based loan approval prediction system developed using Python.

## Features

* Data preprocessing
* Missing value handling
* Exploratory Data Analysis (EDA)
* Feature encoding
* Correlation heatmap
* Feature scaling
* Model training and evaluation
* Feature engineering

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Goal

The goal of this project is to predict loan approval based on applicant financial and personal details using machine learning techniques.
